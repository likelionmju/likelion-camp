ABOUT US

멋쟁이사자처럼은 컴퓨터 비전공자들도 프로그래밍을 할 수 있도록 교육하는 전국 대학 연합 동아리입니다. "내 아이디어를 내 손으로 직접 실현한다."는 목표를 갖고 온/오프라인 강의을 통한 체계적인 프로그래밍을 교육을 진행합니다. 열정과 아이디어만 있다면 누구나 참여 가능한 멋쟁이사자처럼에서 당신의 꿈을 이루어 보세요.

커리큘럼

멋쟁이사자처럼에서는 웹 서비스 런칭을 목표로 프로그래밍 및 창업 교육을 함께 진행합니다. 컴퓨터 비전공자도 쉽게 접근할 수 있는 Python을 중심으로 프로그래밍을 배우며, Django를 활용한 웹 페이지 개발을 진행하고 있습니다.

  • Front-End : HTML5, CSS3, BootStrap / JavaScript(예정)
  • Back-End : Django
  • Programming Language : Python
  • Cooperation Tool : Github
  • Server Deployment : AWS

    • 실제 서비스 기획을 위해 팀 단위로 프로젝트를 진행하게 됩니다. 서비스 아이디어를 구체화시키고, Adobe XD를 활용하여 UI/UX를 설계하는 방법을 배우게 됩니다. 또한 멋쟁이사자처럼 소속의 대학교들과 함께 교류하며 아이디어 공유를 위한 연합 세션을 진행하고 있습니다.

    더 나아가 스타트업 준비를 위해 구체적인 수익 모델 구축 및 회사 운영 방법을 배우고, 다양한 프로젝트 경험을 통해 경쟁력 있는 개발자를 양성하는 것을 목표로 하고 있습니다.
